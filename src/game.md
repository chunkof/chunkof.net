# Game

## PixReversi
**ブラウザゲーム：パズル**

オセロの要領で「お題」と同じパターンをつくるパズルゲーム。オリジナルのお題も作成可能。

**第20回3分ゲーコンテスト参加中。**

### Play
[3分ゲーコンテスト](http://3punge.jp) よりダウンロード(9/23公開予定)

 - 応募番号2012　PixReversi


コンテスト期間中(予定:9/23～10/13)はコンテストページからのみダウンロード可能。
コンテスト終了後は、本サイト上でも公開予定(ダウンロード不要で遊べる)。
